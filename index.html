Batman 2022: Site de Venda de Ingressos

Simulei um site de venda de ingressos para o filme Batman 2022.

O projeto inclui:

  • Funcionalidades: Trailer, formulário de contato, carrossel de imagens, seção de comentários.
  • Tecnologias: HTML5, CSS3, Visual Studio Code e Bootstrap.
  • Destaques: Responsivo, interativo, design moderno.

Desafios e soluções: Criar um layout responsivo que se adaptasse perfeitamente a diferentes tamanhos de tela foi um dos principais desafios. Utilizei o framework Bootstrap e media queries para garantir uma experiência de usuário consistente em todos os dispositivos.
